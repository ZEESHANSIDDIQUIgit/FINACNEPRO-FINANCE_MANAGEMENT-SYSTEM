FinancePro is a full-stack personal finance management application designed to help users track transactions, set visual budget goals, and generate automated financial statements.

Key Features Implemented:
• Authentication & Recovery: Developed a secure user authentication system with signup, login, and a local password recovery workflow.
• Real-Time Tracking & Calculation: Built endpoints to process and calculate real-time income, expense, and net balance states.
• Threshold-Based Budget Monitoring: Integrated dynamic, color-coded spending progress bars (Safe, Warning, Exceeded) to notify users in real-time when limits are approaching.
• Interactive Data Visualization: Embedded Chart.js to render monthly comparisons and category-wise expense distributions.
• Automated Document Compilation: Created a document compile center allowing users to export polished PDF, Excel, and Word statement reports with calculated financial summary blocks.

Technologies Used: Node.js, Express.js, MySQL, JavaScript (ES6), HTML5, CSS3, Chart.js, jsPDF, and SheetJS.
